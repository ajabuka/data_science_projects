### Module
This project was completed in R and the goal was to predict Accident Severity (Traffic Delay) with variables under:
- Traffic Attributes (11 variables)
- Address Attributes (7 variables)
- Weather Attributes (11 variables)
- Point-of-Interest Attributes (13 variables)
- Period of Day (4 variables)
