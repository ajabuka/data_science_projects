# Identifying Bias in Model Development

## This is an assessment project.
## Author: [Jesubukade Emmanuel Ajakaye](https://github.com/ajabuka)

### Description
This is a project is an individual in-course assignment (ICA) that assess:
- Personal and Transferable Skills Development
- Research, Knowledge and Cognitive Skills
- Professional Skills

### Module
This project was completed in Python and the goal was to identify bias in a Model using 3 Fairness Criteria:
- Equal Accuracy
- Group Fairness
- Equal Opportunity

#### Actions
1. Data Cleaning and preprocessing
2. Data Visualization
3. Data Modelling (Random Forest Classifier)
4. Model Optimization
5. Model Performance Evaluation
6. Application of Fairness Criteria
