How to Use This Visualization

This interactive pie chart shows the distribution of transaction amounts across different categories. You can explore the data through multiple levels of detail.

Interactivity Features:

  • Drill Down: Click on any pie segment to see more detailed breakdowns of that category
  • Breadcrumb Navigation: Use the trail at the top to navigate back to previous views
  • Hover Details: Mouse over any segment to see exact transaction counts and amounts
  • Hierarchy Levels: Explore data by Fraud Flag → Transaction Status → Device Used → Transaction Type

Color Coding:

  • Orange: Fraudulent transactions (TRUE)
  • Green: Non-fraudulent transactions (FALSE)
  • Purple: Successful transactions
  • Brown: Failed transactions
  • Other colors represent specific devices and transaction types
