Our Motivation

As data analysts working with transaction monitoring systems, we wanted to understand the behavioral patterns and potential fraud activities across Nigerian states within a key business time frame: 10 AM to 11 PM.

To do this, we analyzed a dataset containing all financial transactions across all Nigerian states within that hour, revealing not just volumes and values, but also hidden fraud patterns.

Data Visualization Journey

1. Exploring State-by-State Trends

We began by mapping out the transaction activity geographically using a choropleth map of Nigeria. This map offered three key layers:

  • Total Transactions per State
  • Total Fraudulent Transactions per State
  • Percentage of Fraudulent to Total Transactions

This revealed fraud hotspots and raised questions about risk concentration.

2. Zooming Into Time-Based Patterns

A time series line chart showed transaction values over the hour, split into:

  • Fraudulent
  • Non-Fraudulent

It revealed subtle fraud spikes shortly after peak legitimate activities.

3. Following the Money by Transaction Type

Using a racing bar chart, we visualized:

  • Cheque Deposit
  • Deposit
  • Withdrawal
  • Transfer
  • ATM Withdrawal
  • Online Transfer
  • Bill Payment
  • POS Withdrawal

This chart showcased dynamic trends and shifting dominance over time.

4. Understanding Device Influence

A box plot compared transaction amounts by device:

  • Mobile
  • Desktop

Revealing which devices were linked to higher value or riskier transactions.

5. Drilling Down with Interactivity

To wrap it up, a multi-layered interactive pie chart provided:

  • Layer 1: Fraud vs. Non-Fraud
  • Layer 2: Transaction Status
  • Layer 3: Device Used
  • Layer 2: Transaction Types

This allowed users to explore the dataset in a meaningful, drill-down experience.
