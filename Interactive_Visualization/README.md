## This project was designed by in Team.

### Description
This is a project is an individual in-course assignment (ICA) that assess:
- Personal and Transferable Skills Development
- Research, Knowledge and Cognitive Skills
- Professional Skills
- Working in a team (cordination, communication, agile system, etc.)

### Module
This project was completed in Javascript, HTML and CSS. d3.js was used to develop visualization on a finance data to detect fradulent transactions. Developed visualizations include:
- Map
- Line Chart
- Racing Bar Chart
- Box Plot
- Drill Down Pie Chart

**Note:** A Dummy data was used for this project to showcase skill formation and development.
